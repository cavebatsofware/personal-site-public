$ Example Document Server

Demonstration of code-gated access control

Built with Rust, Axum, and Alpine Linux

About This Project

This is a demonstration of a secure document server that requires valid access codes to view content. The application is built with modern web technologies and focuses on security and performance.

Backend
Rust + Axum
Container
Alpine Linux
Deployment
AWS ECR + Graviton
Security
Code-gated Access

Features

  • Secure Access Control: Documents only accessible with valid codes
  • Rust Performance: Fast, memory-safe backend
  • Alpine Security: Minimal attack surface (27MB container)
  • Modern Deployment: Automated CI/CD with Makefile
  • Environment Configuration: Easy setup with .env files
  • Health Monitoring: Built-in health check endpoints

Example Implementation

This server can be configured to serve any type of secure document:

  • Personal portfolios and resumes
  • Confidential reports
  • Private documentation
  • Gated content delivery

Technical Architecture

The application uses a multi-stage Alpine Docker build for optimal security and size. Environment-based configuration allows for flexible deployment across different environments.

Security Benefits:

  • Alpine Linux base
  • Static binary compilation
  • Non-root container execution
  • Minimal attack surface
